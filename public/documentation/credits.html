Credits

In the past, we maintained a list of contributors to the Chamilo project along with a description of their contribution. This list is still available here, but as the project grew in size and complexity, maintaining this list became more and more time-consuming to the point we could not maintain it anymore.
Today's contributions are mainly obtainable through several platforms' logs:
  • Code contributions can be found in Github's list of commits for the corresponding branch.
  • Translators contributions can be found on our translations website, but you have to have a valid account to access them.
  • Documentation contributions can be found in each guide's appendix in our online documentation
